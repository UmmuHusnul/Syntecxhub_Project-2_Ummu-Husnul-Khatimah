# Syntecxhub Project 2  
## Landing Page Website

A simple and responsive landing page website created as part of Syntecxhub Project 2.  
This project focuses on clean UI design, smooth navigation, and basic user interaction.

---

## Features
- Responsive design (desktop & mobile)
- Smooth scroll navigation
- Services section
- Contact form connected to Google Sheets
- Social media links

---

## Tech Stack
- HTML5  
- CSS3 (Flexbox & Grid)  
- JavaScript  
- Font Awesome
